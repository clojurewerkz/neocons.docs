An idiomatic, feature rich Clojure client for Neo4J REST API

Feature rich

Neocons supports (almost) all Neo4J REST API features and is constantly tested against bleeding edge server changes, like the Cypher language improvements

Clojuric

Work with Clojure data structures. No unnecessary abstractions. Simple and easy.

Neo4J 2.0 Ready

Provides support for Neo4J 2.0 features: transactions, constraints, labels, and so on.

Batteries included

Neocons provides plenty of minor convenience functions on top of what Neo4J Server REST API provides.

Ready for the cloud

Neocons is ready to be used with the Heroku Neo4J add-on, Cloud Foundry and similar offerings from other PaaS providers.
